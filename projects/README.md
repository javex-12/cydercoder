# Portfolio Projects

This folder contains the actual projects showcased in Michael Dosunmu's portfolio. Each project is a fully functional web application demonstrating different skills and technologies.

## Projects Overview

### 1. TaskMaster - Todo App
**Location:** `todo-app/`
**Technologies:** HTML5, CSS3, JavaScript, LocalStorage
**Features:**
- Task creation, editing, and deletion
- Task filtering (All, Active, Completed)
- Local storage persistence
- Clean, modern interface
- Responsive design

### 2. WeatherNow - Weather App
**Location:** `weather-app/`
**Technologies:** HTML5, CSS3, JavaScript, Weather API, Geolocation
**Features:**
- Real-time weather data
- Location-based forecasts
- 5-day weather predictions
- Geolocation support
- Temperature unit conversion
- Responsive design

### 3. CalcPro - Advanced Calculator
**Location:** `calculator/`
**Technologies:** HTML5, CSS3, JavaScript, Math Functions, LocalStorage
**Features:**
- Basic and scientific calculator modes
- Calculation history with persistence
- Keyboard support
- Trigonometric functions
- Mathematical constants (Ï€, e)
- Error handling and validation

### 4. TechFlow - SaaS Landing Page
**Location:** `landing-page/`
**Technologies:** HTML5, CSS3, JavaScript, Animations
**Features:**
- Modern, conversion-optimized design
- Smooth scroll animations
- Interactive elements
- Responsive layout
- Pricing tables
- Professional styling

Each project is self-contained and can be run by opening the `index.html` file in any modern web browser. No additional setup or dependencies are required.
